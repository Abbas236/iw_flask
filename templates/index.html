Welcome to Library Management System

A complete backend solution for managing books and borrowing records.

Use the navigation menu to browse books or manage borrowing records.

API Documentation
Book Endpoints
  • GET /api/books - List all books
  • GET /api/books/<id> - Get a specific book
  • POST /api/books - Create a new book
  • PUT /api/books/<id> - Update a book
  • DELETE /api/books/<id> - Delete a book
Borrowing Endpoints
  • GET /api/borrows - List all borrowing records
  • GET /api/borrows/active - List active borrowings
  • GET /api/borrows/overdue - List overdue borrowings
  • POST /api/borrow - Borrow a book
  • PUT /api/return/<id> - Return a book
Test API

You can test the API using tools like Postman or via JavaScript fetch in the browser console.

Example JavaScript API call:

fetch('/api/books')
  .then(response => response.json())
  .then(data => console.log(data));
